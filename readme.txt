== Description ==
An AI chatbot that can index your sites pages, posts and other custom post types and allow the user to chat with it.


== Installation ==
This section describes how to install the plugin and get it working.
1. Install ai-chat-with-pages.zip via the Plugins menu

2. Activate the plugin via the Plugins menu

3. Add your OpenAI API Key via the 'AI Chat With Pages' menu

== Frequently Asked Questions ==
= How does the plugin work? =
AI Chat With Pages indexes your sites posts, pages and custom post types into short text chunks in a vector store database. Upon user chat, these are loaded into the conversation as context.
